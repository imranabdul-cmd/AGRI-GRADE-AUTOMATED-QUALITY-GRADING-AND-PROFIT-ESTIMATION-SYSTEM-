# AI Agri Quality Inspection System

## Overview

An AI-powered web application that automates fruit quality inspection using computer vision and deep learning. The system analyzes fruit images, detects defects, assigns quality grades, and estimates profit for agricultural supply chains.

## Features

* AI-based fruit defect detection and grading
* Real-time image processing using Flask APIs
* Interactive dashboard with analytics and history tracking
* Automated quality evaluation reducing manual effort

## Tech Stack

* Python (Flask)
* TensorFlow / Keras
* OpenCV
* HTML, CSS, JavaScript
* SQLite

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open in browser: http://localhost:5000
