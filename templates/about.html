🌾 About AgriVision AI

Smart Agricultural Produce Grading System

AgriVision AI is an intelligent system that uses Artificial Intelligence to classify and grade agricultural produce based on quality parameters and defect density. The application automatically grades fruits, evaluates quality metrics, and calculates revenue projections for industrial packaging and export quality analysis. Our mission is to bring precision and efficiency to agricultural quality control.

🛠️ Technologies Used
  • Python Flask - Web framework for the backend API
  • TensorFlow / Keras - Deep learning framework for AI models
  • OpenCV - Computer vision for image processing
  • SQLite Database - Data persistence and history tracking
  • HTML/CSS/JavaScript - Modern responsive frontend
